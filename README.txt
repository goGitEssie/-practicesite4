PRACTICE SITE FOUR — Esther Gibson

This rebuilt folder follows the Practice Site Four instructions:
1. The folder is named practicesite4 in lowercase.
2. The first header on each page is fixed at the top.
3. The footer is fixed at the bottom on every page.
4. Header and footer use 100% width and declared pixel heights.
5. Wrapper margins account for the fixed header and footer.
6. one.html uses a relative wrapper, absolutely positioned aside/content, pixel widths, and overflow scrolling.
7. two.html uses pixel-based floats, clearfix, and clearing.
8. three.html uses percentage-based floats for a responsive layout.
9. four.html uses inline-block, vertical-align, and percentage widths.
10. five.html uses an inline-block image gallery with object-fit, object-position, borders, margins, shadows, and caption overlays.
11. Extra styling includes CSS patterns, Google Fonts, transitions, and responsive fallbacks.
12. The home page contains a mailto form addressed to egibso5@gmu.edu.
