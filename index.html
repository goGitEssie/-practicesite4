Practice Site Four: CSS Layout Methods

This site demonstrates the layout techniques required for Practice Site Four: fixed and absolute positioning, pixel-based floats, percentage-based responsive floats, inline-block columns, and an inline-block photo gallery.

The first header and footer remain fixed on every page. Wrapper margins keep the page content from being hidden underneath them.
